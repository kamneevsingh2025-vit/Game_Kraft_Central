Setup Instructions:

1.   Download and Extract
   - After downloading the ZIP folder from GitHub, make sure to extract it to your Desktop or another location.
   - Avoid running the files directly from within the ZIP folder.

2.   Ensure Asset Availability
   - The assets folder must be present inside the project’s root directory.
   - Missing assets will cause images not to load properly.

3.   Install Required Dependencies
   - Make sure the requirements.txt file is inside the project folder.
   - Then, install all dependencies using:

     pip install -r requirements.txt

Dependencies:
   - pygame
   - numpy
   - flask
